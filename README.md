
# 2048
### Description:
This is a Python implementation of the popular sliding block puzzle game called **2048**.
Start it with `python3 2048.py`

---
### Controls:
|Key       |Action    |
|:----------:|:----------:|
|W         |Up        |
|A         |Left      |
|S         |Down      |
|D         |Right     |
|Q         |Quit      |
